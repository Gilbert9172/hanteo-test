### 구현 기술 스택

- SpringBoot(3.x)
- Spring Data JPA 
- MySQL
- Docker

<br>

### local-setting 실행 가이드

```
cd local_setting
docker compose up -d
```

추가적으로 기본 제공된 이미지 파일에 있는 데이터는 `./local_setting/mysql/init.sql`에 작성했습니다.
